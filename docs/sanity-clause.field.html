sanity-clause.field package

<< back to sanity-clause index

Field classes that can be used to validate data.

Also contains get-value, deserialize, and validate, which represent the lifecycle of loading data from some other object.

get-value

(FIELD OBJECT &OPTIONAL FIELD-NAME)

Tries to fetch the value corresponding to the field from some datastructure. field-name is used if no attribute is explicitly set on the field.

validate

(FIELD VALUE)

Run the validation checks for a given field and raise a validation-error if it is invalid.

deserialize

(FIELD VALUE)

Converts the value retrieved from the raw data into the datatype the field expects to work with, or fails, raising a conversion-error.
