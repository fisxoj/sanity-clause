sanity-clause.serde package

This package defines the protocol for (de)serializing data to/from other datastructures.

Generic Functions

dump (SCHEMA DATA &OPTIONAL FORMAT) WARNING: This isn't fully tested/supported, but it might work! I reserve the right to change the API in breaking ways, though.

dump takes some data an serializes it to a plist or alist based on format, which can be set to, well, :plist or :alist. .. _generic-function__load:

load (SCHEMA DATA &OPTIONAL FORMAT) Deserializes data into an instance of schema. Fills in default values and validates fields. If schema is :env, it will try to load from environment variables.
